CorpoLang

CorpoLang is the bridge between project managers and programming. It provides the basic functionalities of a general purpose language while being completely written in corporate jargon you'd typically hear in a zoom meeting.

Structure

  • src/: Source code for the CorpoLang interpreter
  • src/interpreter.py: The main interpreter implementation
  • corpolang.py: Main entry point for the language
  • easy_examples/: Simple examples for getting started
  • complex_examples/: Advanced usage examples

Getting Started

To use CorpoLang, import the main module:

import corpolang

Check the examples directory for usage demonstrations.
